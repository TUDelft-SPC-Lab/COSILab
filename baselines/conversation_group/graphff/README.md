# Mingling Group-Formation Benchmark

This repository contains executable benchmark code for group-formation
prediction on the Mingling conversation-group dataset. It provides two
pipelines:

- LSTM/GraphFF, implemented in the repository root.
- DANTE, under `DANTE-master/`.

The repository is intended to contain code, configuration, and documentation.
Large data files, generated benchmark artifacts, model checkpoints, logs, plots,
and result tables are kept outside Git and are distributed separately through
the accompanying data deposit.

## Data Placement

This Git repository does not contain the Mingling data files. After downloading
the external data deposit, place or symlink the benchmark files into the code
repository before running experiments.

Copy the LSTM/GraphFF data from the data deposit:

```text
benchmark/LSTM/mingling1/cam06/  ->  data/mingling1/cam06/
benchmark/LSTM/mingling1/cam08/  ->  data/mingling1/cam08/
benchmark/LSTM/mingling1/cam10/  ->  data/mingling1/cam10/
benchmark/LSTM/mingling2/cam01/  ->  data/mingling2/cam01/
benchmark/LSTM/mingling2/cam03/  ->  data/mingling2/cam03/
```

Copy the DANTE data from the data deposit:

```text
benchmark/DANTE/mingling1/cam06/  ->  DANTE-master/datasets/mingling1/cam06/
benchmark/DANTE/mingling1/cam08/  ->  DANTE-master/datasets/mingling1/cam08/
benchmark/DANTE/mingling1/cam10/  ->  DANTE-master/datasets/mingling1/cam10/
benchmark/DANTE/mingling2/cam01/  ->  DANTE-master/datasets/mingling2/cam01/
benchmark/DANTE/mingling2/cam03/  ->  DANTE-master/datasets/mingling2/cam03/
```

The DANTE fold files in the data deposit are split/data artifacts, not trained
model weights. Model checkpoints are intentionally excluded and can be
regenerated by training.

## Benchmark Scope

The reported benchmark uses five camera-level datasets:

```text
Mingling 1: cam06, cam08, cam10
Mingling 2: cam01, cam03
```

The expected local data layout after downloading the external data package is:

```text
data/
  mingling1/
    cam06/
    cam08/
    cam10/
  mingling2/
    cam01/
    cam03/

DANTE-master/datasets/
  mingling1/
    cam06/
    cam08/
    cam10/
  mingling2/
    cam01/
    cam03/
```

Each LSTM/GraphFF camera directory contains:

```text
features.csv
GT.csv
group_names.txt
scene_continuity.csv
dataset_info.json
```

Each DANTE camera directory contains generated DANTE artifacts:

```text
DS_utils/features.txt
DS_utils/group_names.txt
time_map.csv
coordinates.txt
affinities.txt
timechanges.txt
fold_0/train.p
fold_0/val.p
fold_0/test.p
...
fold_4/train.p
fold_4/val.p
fold_4/test.p
```

Trained model weights are not part of the benchmark data package. They are
training outputs and can be regenerated by running the code with the released
data.

## Environment

The recommended runtime is the Apptainer image defined in
`apptainer/deep_fformation.def`. It contains:

- A PyTorch environment for the LSTM/GraphFF pipeline.
- A TensorFlow 1.x and Keras environment for the DANTE pipeline.

See `apptainer/README.md` for build and HPC usage details.

## LSTM / GraphFF

Run one local fold from the repository root:

```bash
GRAPHFF_DATASET=mingling1/cam06 \
GRAPHFF_TRAIN=1 \
GRAPHFF_DATASET_MAKE=1 \
python main_parallel.py 0
```

Common environment overrides:

```bash
GRAPHFF_FRAME_STRIDE=20
GRAPHFF_SEQ_LEN=10
GRAPHFF_HIDDEN_DIM=8
GRAPHFF_THRESHOLD=1.0
GRAPHFF_NUM_EPOCHS=600
GRAPHFF_BATCH_SIZE=128
GRAPHFF_LR=0.001
GRAPHFF_PATIENCE=50
GRAPHFF_F1_EVAL_EVERY=10
```

Submit one 5-fold camera run on Slurm:

```bash
sbatch --export=ALL,DATASET=mingling2/cam01,TRAIN=1,DATASET_MAKE=1,FRAME_STRIDE=20 \
  slurm/run_vitpose_dataframe_5fold.sbatch
```

Submit all reported LSTM/GraphFF cameras:

```bash
bash slurm/submit_lstm_mingling_all.sh
```

## DANTE

If the external data package already provides the DANTE artifacts, train one
fold from `DANTE-master/deep_fformation` with:

```bash
python run_models.py -d mingling1/cam06 -f 0
```

To regenerate DANTE artifacts from the canonical Mingling camera CSV files,
run from the repository root:

```bash
python DANTE-master/datasets/prepare_mingling.py --session mingling1 --overwrite
python DANTE-master/datasets/prepare_mingling.py --session mingling2 --overwrite
```

Then run the DANTE formatting steps from `DANTE-master/datasets` for each
camera:

```bash
python reformat_data.py -d mingling1/cam06 -p 32 -f 5 -a 6
python build_dataset.py -p mingling1/cam06
```

Submit one 5-fold DANTE camera run on Slurm:

```bash
sbatch --export=ALL,DATASET=mingling2/cam01 slurm/run_dante_mingling_cpu_5fold.sbatch
```

Submit all reported DANTE cameras:

```bash
bash slurm/submit_dante_mingling_all.sh
```

## Result Aggregation

Aggregate LSTM/GraphFF outputs:

```bash
python scripts/aggregate_lstm_mingling_results.py
```

Aggregate DANTE outputs:

```bash
python scripts/aggregate_dante_mingling_results.py
```

The aggregation scripts expect per-fold training outputs under `output/`.
Generated outputs are ignored by Git.


## Repository Policy

The code repository tracks source files, Slurm wrappers, environment recipes,
and README documentation. The following are intentionally excluded from Git:

- Mingling data files.
- DANTE generated dataset artifacts.
- The 4TU upload package.
- Trained model weights.
- Logs, plots, and result tables.
